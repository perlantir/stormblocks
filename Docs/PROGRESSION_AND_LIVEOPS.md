# Progression and Live Ops

## Free-first progression

The game should be generous and free.

Progression rewards:

- Cosmetics.
- Titles.
- Profile frames.
- Camp decorations.
- Survivor outfits.
- VFX variants.
- Achievement badges.

No paid power.

## Daily Storm

Daily Storm is the retention engine.

Daily result tracks:

- Score.
- Survivors rescued.
- Best combo.
- Storm tiles destroyed.
- Clutch saves.
- Streak.

## Weekly Tempest Trials

Weekly mode with 5 challenge runs.

Rewards:

- Cosmetic badge.
- Profile banner.
- Camp decoration.

## Seasons

Cosmetic-only seasons can be added later without paid power.

Season examples:

- Lantern Season.
- Snow Shelter.
- Neon Tempest.
- Cozy Campfire.
- Sky Rescue.
