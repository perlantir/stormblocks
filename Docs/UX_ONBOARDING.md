# UX and Onboarding

## First 10 seconds

No text-heavy tutorial.

Flow:

1. Show 8x8 board with warm camp center.
2. Show storm at outer edge.
3. Show three pieces at bottom.
4. Highlight one piece.
5. Show ghost placement.
6. Player drags piece.
7. A line clears.
8. Survivors run to camp.
9. Storm pushes back.
10. “Saved!” appears briefly.

The player should understand the game before seeing a tutorial box.

## Tutorial philosophy

Teach only one concept at a time.

- First: place blocks.
- Second: clear line.
- Third: rescue survivors.
- Fourth: storm spreads.
- Fifth: pushback.
- Sixth: combos.
- Seventh: Daily Storm.

## UI principles

- Top HUD: score, rescued, best/daily.
- Board center.
- Piece tray bottom.
- Pause/settings small.
- No giant push button.
- No hint button in first-run tutorial.
- Undo can appear later as an earned comfort tool.
