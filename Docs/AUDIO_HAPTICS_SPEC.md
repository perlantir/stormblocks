# Audio and Haptics Spec

## Audio style

Warm, satisfying, magical storm-rescue.

## Required sound events

- UI tap.
- Piece pick up.
- Piece hover.
- Valid placement.
- Invalid placement.
- Line clear.
- Multi-line combo.
- Survivor rescued.
- Storm warning.
- Storm spread.
- Storm Pushback.
- Clutch save.
- Near-death loop intensity.
- Game over.
- Results celebration.
- Cosmetic unlock.
- Daily Storm start/end.

## Haptics

- Light tap: UI and piece pickup.
- Medium: valid placement.
- Warning pulse: storm spread preview.
- Heavy/impact: line clear.
- Success burst: combo/pushback.
- Long warning: near-death.

Haptics must be optional in settings.
