# Storm Blocks Support

Storm Blocks is a fair casual puzzle game by UBER KIWI LLC.

For support, bug reports, gameplay feedback, accessibility issues, or feature requests, contact:

- Email: support@perlantir.com
- Project support tracker: https://github.com/perlantir/stormblocks/issues

Please include:

- Device model and iOS version.
- App version.
- A short description of what happened.
- Steps to reproduce the issue, if known.

Storm Blocks does not use forced ads, paid power, paid speedups, gacha, or loot boxes.
