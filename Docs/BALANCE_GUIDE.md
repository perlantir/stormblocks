# Balance Guide

## Initial tuning targets

Endless run average for new player:

- First run: 60-120 seconds.
- Skilled run: 5-10 minutes.
- Great run: 10+ minutes.

## Storm timing

Base curve:

- Calm: spread every 4 placements.
- Strategic: spread every 3 placements.
- Panic: spread every 2 placements.

## Pushback

Pushback should be strong enough to create comebacks but not erase all danger.

Suggested starting rule:

- Clearing a line with storm tiles destroys storm tiles in the cleared line.
- Combo clears apply bonus pushback to adjacent storm tiles.
- Clutch clears add extra pushback if storm is near camp.

## Scoring config

Starting values:

- Cleared cell: 10.
- Line clear: 100.
- Survivor rescued: 25.
- Storm tile destroyed: 50.
- Combo multiplier: 1x, 2x, 3x, 5x, 8x.
- Clutch save: 500+.
- Perfect set: 250.

These values should be stored in config, not hard-coded.
