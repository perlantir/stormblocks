# Design References

These images are guidance for the final visual direction. They are not exact UI specs.

## Usage

- `04_ref_primary_gameplay_layout.jpg`: primary gameplay target.
- `02_ref_mass_market_charm.jpg`: mass-market charm, colors, character warmth.
- `03_ref_storm_intensity_combo.jpg`: near-death and combo/pushback drama.
- `01_ref_logo_mood_phone_frame.jpg`: logo/title atmosphere.

## Important interpretation

Do not copy every UI element literally.

The reference images include buttons such as PUSH, HINT, UNDO, and CLEAR ROW. These may look nice in a concept image but should not override the game design.

For production:

- Storm Pushback is automatic.
- No giant push button in core gameplay.
- No hint button in first-run flow.
- Undo is optional and introduced later.
- Gameplay must stay readable.
