# Screen Prompts

## Main Gameplay Calm

Create a production-quality portrait iPhone gameplay screen for “Storm Blocks,” a mainstream premium casual 3D block puzzle game.

Scene: The player is in the middle of an endless run. The 8x8 grid is centered. A warm tiny rescue camp glows in the middle. Colorful soft 3D block pieces are placed on the grid. Blue-purple storm tiles are creeping inward from the edges. Tiny cute survivors are visible near safe tiles. The bottom tray shows three draggable block pieces. The top HUD shows score, rescued survivors, and best score.

Make it look like a top-charting mobile puzzle game with an original storm-rescue identity.

## Near-Death State

Create a production-quality portrait iPhone gameplay screen for “Storm Blocks” in a near-death state. The storm has advanced close to the central camp. Camp pulses with danger. Screen edges have subtle storm vignette. Wind and lightning intensify. A comeback move is clearly available. The board remains readable.

Emotional goal: “I am almost dead, but one smart move can save this run.”

## Storm Pushback Combo

Create a production-quality portrait iPhone gameplay screen for “Storm Blocks” showing a huge automatic Storm Pushback combo moment. Cleared rows/columns glow. Blue-purple storm tiles are blasted backward by a warm golden rescue wave. Tiny survivors run toward camp. The camp glows. Colorful toy blocks sparkle. HUD shows combo, rescued count, and score.

Emotional goal: heroic, juicy, polished, addictive.

## Daily Storm

Create a production-quality portrait iPhone Daily Storm screen. Header: Daily Storm. Subtext: Same storm. Same pieces. Beat today’s score. Include preview card, streak, best daily score, start button, and friend/local leaderboard preview.

## Results

Create a production-quality portrait iPhone results screen after a run. Show survivors rescued, score, best combo, storm tiles destroyed, clutch saves, daily rank/local best, and buttons for Play Again, Daily Storm, and Share.
